This endpoint returns the OpenAI-compatible tool schema that you register with your LLM to enable it to invoke PureFrame capabilities. Fetching this schema at runtime ensures your agent always uses the latest tool definitions — including parameter names, types, and descriptions — without requiring code changes on your side when PureFrame updates its toolset.

## Endpoint

```
GET https://api.pureframe.ai/v1/agent/schema.json
```

**Authentication required:** No

<Note>
  This endpoint is public. Any client can fetch the schema without an API key.
</Note>

## Available Tools

The schema includes definitions for the following PureFrame agent tools:

| Tool | Description |
|---|---|
| `search_videos` | Perform semantic search across videos in a collection using a natural-language query. |
| `list_collections` | List all video collections available in your PureFrame account. |
| `get_collection` | Retrieve metadata and details for a specific collection by ID. |
| `get_video` | Retrieve metadata and indexed frame information for a specific video by ID. |

## Request Example

```bash
curl https://api.pureframe.ai/v1/agent/schema.json
```

## Response

Returns `200 OK` with an OpenAI-compatible `tools` array. Each entry in the array follows the [OpenAI function calling schema](https://platform.openai.com/docs/guides/function-calling) format and can be passed directly to the `tools` parameter of a chat completion request.

```json
{
  "tools": [
    {
      "type": "function",
      "function": {
        "name": "search_videos",
        "description": "Search for videos semantically matching a natural-language query within a collection.",
        "parameters": {
          "type": "object",
          "properties": {
            "query": {
              "type": "string",
              "description": "Natural-language search query, e.g. 'person waving at camera'."
            },
            "collection_id": {
              "type": "string",
              "description": "The ID of the collection to search within."
            }
          },
          "required": ["query", "collection_id"]
        }
      }
    },
    {
      "type": "function",
      "function": {
        "name": "list_collections",
        "description": "List all video collections in the account.",
        "parameters": {
          "type": "object",
          "properties": {}
        }
      }
    },
    {
      "type": "function",
      "function": {
        "name": "get_collection",
        "description": "Get metadata for a specific collection by ID.",
        "parameters": {
          "type": "object",
          "properties": {
            "collection_id": {
              "type": "string",
              "description": "The unique identifier of the collection."
            }
          },
          "required": ["collection_id"]
        }
      }
    },
    {
      "type": "function",
      "function": {
        "name": "get_video",
        "description": "Get metadata and indexed frame information for a specific video.",
        "parameters": {
          "type": "object",
          "properties": {
            "video_id": {
              "type": "string",
              "description": "The unique identifier of the video."
            }
          },
          "required": ["video_id"]
        }
      }
    }
  ]
}
```

For a full walkthrough of registering this schema with your LLM and handling tool calls, see the [AI Agent Integration guide](/guides/ai-agent-integration).
