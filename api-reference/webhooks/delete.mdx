Permanently delete a webhook endpoint from your PureFrame account. Once deleted, PureFrame stops delivering events to the associated URL immediately — no further payloads will be sent, and the endpoint cannot be restored. If you only want to pause delivery temporarily, consider setting `is_active` to `false` via the [Update Endpoint](/api-reference/webhooks/update) endpoint instead.

## Endpoint

```
DELETE https://api.pureframe.ai/v1/webhooks/{endpoint_id}
```

**Authentication required:** Yes

## Path Parameters

<ParamField path="endpoint_id" type="string" required>
  The unique identifier of the webhook endpoint to delete (e.g. `wh_abc123`).
</ParamField>

## Request Example

```bash
curl -X DELETE https://api.pureframe.ai/v1/webhooks/wh_abc123 \
  -H "Authorization: Bearer pf_your_key"
```

## Response

Returns `204 No Content` on success. The response body is empty.

```
HTTP/1.1 204 No Content
```
