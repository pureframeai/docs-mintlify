PureFrame is a video intelligence API that makes your video library fully searchable. Upload any video, let PureFrame index every frame and transcript, then query it with natural language or a reference image to instantly retrieve the exact timestamped clips you need.

## How PureFrame works

<Steps>
  <Step title="Create a collection">
    Organize your videos into named collections. A collection is a searchable library — create one per project, customer, or content category.
  </Step>
  <Step title="Upload and index videos">
    Upload video files in any common format (MP4, MOV, AVI, WebM, MKV). PureFrame processes each video asynchronously, indexing visual frames, scenes, and transcripts.
  </Step>
  <Step title="Search with text or images">
    Query your collection using a text description or a reference image. PureFrame returns ranked, timestamped clips with presigned video URLs and thumbnails.
  </Step>
  <Step title="Integrate anywhere">
    Use search results in your app, feed thumbnails to vision models, or hook into processing events via webhooks.
  </Step>
</Steps>
